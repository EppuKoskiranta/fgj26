# Lotion pot

Ingredients are added to the lotion pot.
The lotion pot has the absolute values for lotion effects.

Lotion pot has:
- getRatio() which returns the ratio of the lotion effects of the currently stored ingredients.

# Mixing gameplay

Move the mixing spoon to mix the ingredients together.
Ingredients should be separate and mixed together gradually.
At the end of the mixing step: create the lotion object

# Lotion object

The lotion pot has the absolute values for lotion effects.

Is created when the pot is interacted with after the mixing step.

Lotion has:
- getRatio() which returns the ratio of the lotion effects of the currently contained ingredients.
